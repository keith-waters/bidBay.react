## bidBay
### a tiny bidding app using Firebase

## To try it out:

 - clone the repository
 - run npm install
 - create a config.js file in the /src/firebase directory
 - format the file: 
 module.exports = {
   YOUR_FIREBASE_CREDENTIALS
 } 
 - run npm start


## Or try the demo here:
https://bidbay-11e97.firebaseapp.com/
